# 🔍 Algoritmos de Ordenação e Busca em Java

Este repositório contém a implementação e análise de performance de algoritmos clássicos de Ciência da Computação, utilizando uma estrutura de gerenciamento de candidatos para um concurso musical.

## 📋 Funcionalidades
O projeto demonstra como manipular vetores de objetos complexos, aplicando diferentes estratégias de organização e localização de dados:

### 🛠️ Algoritmos de Ordenação
* **Bubble Sort**: Ordenação por nome (estratégia de trocas adjacentes).
* **Selection Sort**: Ordenação pelo número do candidato (seleção do menor valor).
* **Insertion Sort**: Ordenação por idade (inserção ordenada em sublista).

### 🔎 Métodos de Busca
* **Busca Sequencial**: Localização de dados em listas não ordenadas.
* **Busca Binária**: Localização otimizada em listas previamente ordenadas, reduzindo drasticamente o número de comparações.

## 🚀 Diferenciais do Projeto
* **Análise de Eficiência**: O código imprime no console a quantidade de trocas e comparações realizadas, permitindo visualizar a complexidade de cada algoritmo na prática.
* **Manipulação de Objetos**: Uso de `Arrays.copyOf` para preservar a integridade dos dados originais.
* **Encapsulamento**: Uso rigoroso de classes Java (POO) com atributos privados e métodos acessores.
